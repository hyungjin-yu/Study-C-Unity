**모르는게 생길 때의 대처법**
1.   **문제 기반 검색 시스템 만들기**
    - 옵시디언에서 `Ctrl+O` (파일 검색) 또는 `Ctrl+Shift+F` (전체 검색) 활용
    - 예: "충돌 처리"로 검색 → 관련 노트 즉시 찾기 
2. **실전 프로젝트 노트 별도 생성**
    - `프로젝트명 - 문제 해결 로그` 형식
    - 문제 발생 → 옵시디언 검색 → 해결 → 로그 기록
    - 이론 노트에 `[[프로젝트 링크]]` 추가
3. **실습 먼저, 이론은 참고용**
    - 작은 게임부터 만들기 시작
    - 막힐 때만 옵시디언 검색
    - "이건 어떻게 하지?" → 검색어: "Raycast", "코루틴" 등
4. **Quick Reference 노트 생성**
    - 자주 쓰는 코드 스니펫만 모은 치트시트
    - 한 페이지에 핵심만 요약

**핵심:** 이론 100% 숙지 후 개발 ❌ → 개발하면서 필요할 때 찾아보기 ✅
